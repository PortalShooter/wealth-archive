# simpleEaseObjects

Аналогично предыдущему, но работает с объектами.  
Не работает с объектами с вложенностью! (можно пройтись рекурсией, но надо ли.. безопасней оставить неглубокие объекты)  
**Синтаксис**  
```javaScript
a1 = simpleEaseArrays(a1, a2, 20);
```
a1: Object (shallow)  
a2: Object (shallow)
